# 🎁 Buscador de Regalos (Recursivo con Ternarios e If)

Este programa en JavaScript busca un regalo dentro de una lista usando **recursividad**, y combina **operadores ternarios** con condicionales **`if`** para controlar el flujo del programa de manera clara y eficiente.

---

## 🧠 ¿Cómo funciona?

El programa recorre una lista de regalos uno por uno utilizando **recursividad**. En cada paso:

- Usa `if` para mostrar mensajes por consola (`console.log`) que no necesitan retornar nada.
- Usa **operadores ternarios** para las decisiones principales del flujo, que sí deben devolver un valor.
